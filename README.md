# demo-ingitdb

A demo IngitDB database - versioned, with support for multiple languages and views

## Collections:

- [GEO](geo)
    - [countries](geo/countries)
        - states
            - cities
- [ToDo](todo)
    - [tasks](todo/tasks)
    - [tags](todo/tags)
    - [statuses](todo/statuses)

## UI for browsing and editing

You can browse and edit data using GitHub but for advanced features, consider using DataTug.app.

### DataTug.app

    - Web UI: https://datatug.app/#ingitdb=github.com/ingitdb/demo-ingitdb
    - CLI: install https://github.com/datatug/datatug
